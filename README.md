# 🏙️ Luxury Housing Sales Analysis — Bengaluru

An end-to-end real estate analytics pipeline built for the Bengaluru luxury housing market.

**Stack:** Python · SQLite · Streamlit · Plotly

---

## 📁 Project Structure

```
luxury-housing-bengaluru/
├── app.py                          # Streamlit dashboard (main app)
├── clean_and_load.py               # ETL script — cleaning + DB loading
├── sql_queries.sql                 # SQL validation & analytics queries
├── requirements.txt                # Python dependencies
├── Luxury_Housing_Bangalore.csv    # Raw dataset (add manually — not in repo)
├── luxury_housing.db               # SQLite database (generated by ETL script)
└── luxury_housing_clean.csv        # Cleaned dataset (generated by ETL script)
```

---

## 🚀 How to Run

### 1. Clone the repo
```bash
git clone https://github.com/YOUR_USERNAME/luxury-housing-bengaluru.git
cd luxury-housing-bengaluru
```

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

### 3. Add the dataset
Place `Luxury_Housing_Bangalore.csv` in the project root folder.

### 4. Run the ETL pipeline
```bash
python clean_and_load.py
```
This generates:
- `luxury_housing.db` — SQLite database
- `luxury_housing_clean.csv` — cleaned 100k-row dataset

### 5. Launch the dashboard
```bash
streamlit run app.py
```
Open your browser at `http://localhost:8501`

## 📊 Dashboard Features

| Visual | Insight |
|--------|---------|
| Quarterly Trend (Line) | How bookings move across Q1–Q4 per micro-market |
| Configuration Demand (Donut) | Most popular unit types (3BHK / 4BHK / 5BHK+) |
| Developer Performance (Bar) | Revenue and avg ticket size per builder |
| Booking Conversion (Bar) | Which micro-markets convert best |
| Sales Channel Efficiency (Stacked) | Broker vs Direct vs Online vs NRI Desk |
| Amenity vs Booking Rate (Scatter) | Does amenity score drive conversions? |
| Possession Status × Buyer Type | What stage do HNI / NRI buyers prefer? |
| Developer × Quarter Matrix (Heatmap) | Quarterly revenue dominance by builder |
| Top 5 Developers (Table) | Revenue + booking success leaderboard |
| NRI Buyer Share (Bar) | NRI concentration by micro-market |

All visuals respond to **4 sidebar filters**: Developer, Fiscal Quarter, Micro-Market, Configuration.

## 🧹 Data Cleaning Summary

| Issue | Fix |
|-------|-----|
| 1,000 duplicate rows | `drop_duplicates()` |
| `Ticket_Price_Cr` stored as string | regex strip + float cast |
| ~500 rows with `Unit_Size_Sqft = -1` | Replace with NaN → group median fill |
| 10,090 null `Amenity_Score` | Fill with global median (7.50) |
| 10,019 null `Ticket_Price_Cr` | Fill with per-Developer median |
| 10,046 null `Unit_Size_Sqft` | Fill with per-Configuration median |
| Inconsistent text casing | `.str.strip().str.title()` |

**Result:** 100,000 clean rows · 25 columns · 0 nulls

## ⚙️ Feature Engineering

| Column | Logic |
|--------|-------|
| `Booking_Flag` | 1 if Transaction_Type == Primary |
| `Purchase_Quarter_Label` | Indian fiscal quarter (Q1 FY24, Q2 FY25…) |
| `Price_per_Sqft` | (Ticket_Price_Cr × 10M) / Unit_Size_Sqft |
| `Amenity_Band` | Low / Medium / High / Premium buckets |
| `NRI_Flag` | 1 if NRI_Buyer == yes |

## 💡 Key Business Insights

- **Jayanagar** leads in volume (6,379 units); **Sarjapur Road** has the highest avg price/sqft (₹23,307)
- **Prestige Group** tops revenue at ₹1,16,718 Cr across 9,224 units
- Booking conversion is ~50% across all sales channels — channel doesn't significantly differentiate
- Demand is evenly split: 5BHK+ (33.5%), 3BHK (33.4%), 4BHK (33.1%)
- Q4 FY24 had peak volume but lowest avg price — possible seasonal discounting
